# AXTP Demo Docs

| 文档 | 说明 |
|---|---|
| [ota-stream.md](ota-stream.md) | OTA 场景下 RPC 控制面、STREAM 数据面和 CONTROL 可靠性的端到端示例。 |
| [mvp-normative.md](mvp-normative.md) | AXTP v1 MVP 链路的规范化端到端示例。 |

历史或已被替代的 demo 文档放在 [`docs/archive/demo/`](../archive/demo/)；不要从 archive demo 推导当前协议事实。
